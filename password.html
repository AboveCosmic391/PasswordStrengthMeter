Password Strength Meter

In this project I wanted to create a password strength mater. Features include:

  • Prints the password as the user types
  • Prints the strength percentage as the user types
  • Instructions for the end user
  • A progress bar that increments as the strength value changes
  • A requirement to type a confirmation password
  • Confirmation password must match the original password
  • Button click checks all field and "registers" the user
